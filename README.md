
# Playwright Automation Framework (POM + TypeScript)

## 📌 Overview
This project demonstrates an end-to-end test automation framework using Playwright with TypeScript and Page Object Model (POM) design pattern.

## 🚀 Tech Stack
- Playwright
- TypeScript
- Node.js
- Page Object Model

## 📂 Project Structure
- tests → Test scripts
- pages → Page Object Models
- fixtures → Test data
- utils → Configurations

## ✅ Test Scenario
- Login to SauceDemo
- Add products to cart
- Complete checkout
- Verify order success

## ▶️ Run Tests
```bash
npm install
npx playwright install
npm test
``
